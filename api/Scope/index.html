Scope

This item is only intended to be used by the module's authors. Private

The Scope class is used internally by LuauDispatcher during execution of ASTs to manage variables. It cannot be directly accessed unless you specifically require it, which I don't know why you'd ever do but you do you

Functions​

declare​

</>
Scope:declare(
name: string,
value: any
) → ()

Declares a new local variable in a scope. Even if a var exists with the same name elsewhere it will be re-declared.

get​

</>
Scope:get(name: string) → (
value: any,
found: boolean
)

Searches for a given variable and returns it's value, and also searches all elevated scopes. If no variable of that name exists, returns nil and found=false
