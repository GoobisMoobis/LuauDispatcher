Runtime

This item is only intended to be used by the module's authors. Private

The Runtime class is used internally by LuauDispatcher to execute ASTs. It can be directly interacted with via Dispatcher.Runtime.

There is really no reason to ever manually use this class, however the option is open if you want to do so.
